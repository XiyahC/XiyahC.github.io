#【推薦】hexo-generator-index從 2.0.0 開始，已經支持文章置頂功能。你可以直接在文章的front-matter區域裏添加sticky: 1屬性來把這篇文章置頂。數值越大，置頂的優先級越大。
title: {{ title }}
date: {{ date }}
# 选填
updated: #updated date 
tags: #tags
categories: #categories }}
keywords: #keywords }}
description: #description }}
top_img: #文章顶部图片 }}
comments: #显示文章评论，默认true }}
cover:  #文章缩略图，如果没有设置top_img，文章页顶部将显示缩略图 }}
toc: #显示文章toc }}
toc_number: #显示tocnumber }}
toc_style_simple: #显示toc简洁模式 }}
copyright: #文章版权显示，默认enable }}
copyright_author: #版权作者 }}
copyright_author_href: #版权链接 }}
copyright_url: #版权链接 }}
copyright_info: #版权文字 }}
mathjax: #mathjax设置，默认false }}
katex: #显示katex，默认false }}
aplayer: #在需要的页面加载aplayer的js和css }}
highlight_shrink: #是否展开代码框 }}
aside: #默认true，显示侧边栏 }}
